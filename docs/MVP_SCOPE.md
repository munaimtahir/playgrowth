# MVP Scope

## Included

- App profile management
- Daily metric tracking
- CSV imports for metrics and reviews
- Dashboard with bottleneck diagnosis
- Weekly and manual growth reports
- Recommendation queue with manual statuses
- Review analysis and review themes
- Listing advisor drafts
- Experiment planning
- Manual action logging
- Audit logging

## Excluded

- Play Console writes
- Google Ads writes
- Production releases
- Review response automation
- Fake reviews or installs
- Keyword stuffing
- Misleading app claims
- Aggressive dark-pattern monetization

## Status values

Recommendations use:

- `new`
- `accepted`
- `rejected`
- `done`
- `monitoring`

## Manual workflow

1. Import data or seed demo data.
2. Generate a report.
3. Review the bottleneck diagnosis.
4. Accept one or more recommendations.
5. Make the change manually outside the app.
6. Log the action.
7. Review outcomes later.
